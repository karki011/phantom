Download for macOS
Capabilities

Everything you need, nothing you don't.

A focused set of tools designed around how developers actually work.

Multi-Worktree Management

Work across multiple git worktrees simultaneously. Switch contexts without stashing — each worktree is an independent workspace.

Integrated Terminal

Full-featured terminal sessions embedded in your workspace. Run commands, watch logs, and manage processes without leaving the app.

AI Chat with Claude

Built-in AI assistance powered by Claude. Get help with code, generate commits, review changes, and accelerate your development workflow.

Developer Journal

Track your development sessions with an integrated journal. Capture thoughts, decisions, and progress as you work.

Achievement System

Earn achievements as you hit development milestones. A gamified layer that makes shipping code even more satisfying.

Project Orchestration

Session tracking, CI/CD status, PR management, and branch activity — all surfaced in real-time through a unified dashboard.
